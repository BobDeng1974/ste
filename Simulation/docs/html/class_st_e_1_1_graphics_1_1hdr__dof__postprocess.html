Set the camera aperture parameter. Those parameters affect the depth of field of the resulting image.

Parameters
diameterLens diameter in world units. Defaults to human eye pupil diameter which ranges from 2e-3 to 8e-3.
focal_lengthFocal length world units. Defaults to human eye focal length, about 23e-3.
