Detailed Description

template<typename Specialization, typename Descriptor, template< class > typename Storage>
class StE::Core::resource_storage< Specialization, Descriptor, Storage >

resource_storage base class

resource_storage defines GPU storage of resources supporting updates and deletes.

Parameters
SpecializationSubclass type (CRTP).
DescriptorDescriptor type. Descriptor is the POD that will be uploaded to the GPU for each resource.
